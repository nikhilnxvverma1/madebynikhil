--- title: "Event Handling" date: 2015-12-24T12:59:48+05:30 draft: true --- Upto this point, we were creating our user interfaces without any user interactions in them. In this tutorial, we will cover the most important aspect of any user interface i.e. event handling. Before I go forwards, I want to clarify the term "event" in this context. An event can be anything that occurs as part of an ongoing system. Very specifically, "user event" are the events that happen as part of user's interaction with the interface. These include, but are not limited to mouse events (like mouse pressed,mouse dragged, and mouse released events), keyboard events (like key pressed, key released etc) and even scroll events (like scroll started, scrolling, scroll finished). General Button based event handling Buttons present the most common scenario for handling events.
